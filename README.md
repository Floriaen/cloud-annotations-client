# Cloud Annotations Client
This is the setup and usage information for a local development environment for the Cloud Annotations tool.

## [Try it now](https://cloud.annotations.ai)

## Project setup
`git clone` the repo and `cd` into it by running the following command:

```bash
git clone https://github.com/bourdakos1/cloud-annotations-client.git
cd cloud-annotations-client
```

Install the server dependencies:
```
npm install
```

Install the client dependencies:
```
cd client && npm install
````

> **Note: Youâ€™ll need to have Node 8.10.0 or later on your local development machine.** You can use [nvm](https://github.com/creationix/nvm#installation) (macOS/Linux) or [nvm-windows](https://github.com/coreybutler/nvm-windows#node-version-manager-nvm-for-windows) to easily switch Node versions between different projects.

## Local development
Start the app:
```
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
